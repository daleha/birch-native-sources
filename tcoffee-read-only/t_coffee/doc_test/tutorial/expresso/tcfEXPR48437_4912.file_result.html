How To Read the Results ?

msf aln, clustal aln, fasta aln: MSAs in the suggested formats.
score_html: Local MSA reliability using the CORE index. Red and orange portions are considered correctly aligned.
score_ascii: Coded version of the score_html file.
dnd file: Guide tree (Newick format). It's NOT a Phylogenetic Tree!
ph file: NJ Phylogenetic tree.
template file: Displays the template found.
